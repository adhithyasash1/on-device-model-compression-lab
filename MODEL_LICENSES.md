# Model Sources

## MobileViT

V1 targets Hugging Face model IDs:

- `apple/mobilevit-xx-small`
- `apple/mobilevit-small`
- `apple/mobilevit-x-small`

Weights are downloaded locally by the user through the normal Hugging Face and Transformers cache paths. Model weights are not committed to this repo.

The repository code is MIT licensed. The Apple MobileViT weights are separate upstream assets and remain under their upstream model terms. Hugging Face currently lists these MobileViT model cards with `License: other`; the model license file for `apple/mobilevit-small` describes the Apple Machine Learning Research Model License Agreement and limits use to research purposes. Treat compressed Core ML outputs as derived model artifacts, not MIT-licensed project code.
